# McServerWatch
# 🛡️ ServerWatch v1.0

A secure, open-source Minecraft server status tracker built with Python and CustomTkinter. 

This project completely avoids false-positive virus detections by delivering raw, transparent code instead of sketchy, compiled `.exe` files.

## 🚀 Features
* **Modern UI:** Clean dark theme powered by CustomTkinter.
* **Lag-Free Queries:** Uses Python's threading library to prevent application freezing during server checks.
* **Smart Batch Launcher:** The `.bat` script automatically manages library installations in the background without terminal spam.
* **Independent Execution:** You can safely close the CMD window; the main panel will keep running smoothly.

## 📦 How to Use
1. Click the green **Code** button at the top right and select **Download ZIP** (or clone the repository).
2. Extract the files and double-click `ServerWatch_Calistir.bat`.
3. The script will check and automatically install required libraries (`customtkinter`, `mcstatus`) on its first run, then launch the panel.
